Project Description

The Directorate for Applied Short-Term Programmes (DASP) at Mahatma Gandhi University, committed to bridging academic learning with industry demands, has developed a cutting-edge pothole detection system that exemplifies applied research and skill development. This system leverages deep learning and advanced image processing techniques to accurately identify and classify potholes, enhancing infrastructure monitoring and maintenance. The user-friendly frontend is built using Angular, delivering an interactive experience, while the backend, developed with Python and the Flask framework, ensures efficient data processing and system scalability. Integration with AWS S3 facilitates secure storage and seamless API-based communication, supporting real-time data access and analysis.
